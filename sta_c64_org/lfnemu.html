LFN drivers

last updated on 2017-10-27

Long file name drivers allow you to access long file names under real DOS and in the DOS shells of Windows NT 4.0.

DOS

Long file name drivers for DOS

  • DOSLFN 0.41c (2012-12-11), LFN driver for DOS (© by Henrik Haftmann, 2001-2003, and Jason Hood, 2003-2012) [doslfn.zip, 306.175 bytes] (homepage)

    • NOTE: Along with SHSUCDX, this software can also access long file names on CD's/DVD's.

  • OpenDOS LFN Support Beta 3 (1997-11-24), LFN driver for DOS (© by Caldera Inc., 1997) [lfnb3.zip, 71.898 bytes]

    • WARNING: In its original form, OpenDOS LFN Support runs under Caldera OpenDOS only. See the fix below for a patch that removes the DOS version check so that you can run it under other DOS'es, as well.

    • MORE: You may also want to apply the fix to OpenDOS LFN Support (© by Vsevolod Volkov, 1998) [lfnb3fix.zip, 1.774 bytes]

  • DRFAT32 R1.00 (2001-09-30), FAT32 disk driver and LFN driver for DOS (© by Caldera Inc., 1997-2001) [drfat32.zip, 28.897 bytes]

  • LFNDOS 1.06 (1999-08-29), LFN driver for DOS (© by Chris Jones, 1998-1999) [lfndos.zip, 24.494 bytes]

    • SOURCE: You may also want to check out the source of LFNDOS [lfndssrc.zip, 124.392 bytes]. Also includes the source of an unreleased 1.07 alpha 2 [lfndsnew.zip].

  • Odi's LFN Tools 1.79 (2003-08-14), utilities to access long file names under DOS (© by Ortwin Glück, 1999-2003) [lfn-1.79-en.zip, 284.555 bytes] (homepage)

    • NOTE: This is software not a driver.

  • Star LFN, adds long file name support to real DOS (see the homepage)

Long file name-related software for DOS

  • LFNSORT 1.5 (1997-04-26), directory sort software with LFN support (© by D. J. Murdoch, 1995-1997) [lfnsrt15.zip, 34.163 bytes] (archived homepage)

Windows NT

Long file name drivers for Windows NT

  • LFN/NT 1.0 beta 3 (2000-10-15), LFN driver for the DOS shell of Windows NT (© by MultiBug Software, 2000) [lfnnt.zip, 7.854 bytes]

  • NTLFN 0.81.71 (2016-01-16), LFN driver for the DOS shell of Windows NT (© by Wojciech Galazka and Joe Forster/STA, 1999-2016) [ntlf081b.zip, 40.815 bytes] (archived original homepage)

    • SOURCE: You may also want to check out the source of NTLFN [ntlf081s.zip, 95.972 bytes]

    • PREVIOUS: binary [ntlfn08b.zip, 25.005 bytes] and source [ntlfn08s.zip, 64.443 bytes] of NTLFN 0.8.

  • nt-lfn 1.001 (1998-09-30), LFN driver for the DOS shell of Windows NT (© by Andrew Crabtree, 1998) [lfn1001.zip, 14.320 bytes]

    • WARNING: Obsolete, use NTLFN instead.

    • SOURCE: You may also want to check out the source of nt-lfn [lfnd1001.zip, 67.785 bytes]
