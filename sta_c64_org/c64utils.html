C64 utilities

last updated on 2015-06-26

Here are lots of useful utilities for your Commodore 64, some of them written by me. The STA disk is actually two disk images with a compilation of my favorite utilities.

Downloads

  • The STA disk, side 1 (1995-02-08) [stadisk1.zip, 142.045 bytes]

    • Menu Load, loads the directory of a disk and then fastloads the software selected

      • USAGE: Load with "...,8,1" and it will autostart.

      • ALTERNATE: Use Menu Load.NTSC and Menu Load.DTV1 (on side 2) instead on NTSC and C64-DTV version 1 machines, respectively.

    • Read Me, notes about the software in the compilation

      • LANGUAGE: Hungarian.

    • Star Turbo.32000, disk turbo that speeds up LOAD/VERIFY and SAVE to 6-8 times the original speed

      • USAGE: Start with SYS 32000.

      • ALTERNATE: Use Star Turbo.NTSC and Star Turbo.DTV1 instead on NTSC and C64-DTV version 1 machines, respectively.

    • Star Turbo.52992, Star Turbo

      • USAGE: Start with SYS 52992.

    • Star Turbo Relocator

      • USAGE: Start with SYS 2064,xxxx and you'll get a runnable version of Star Turbo to be started with SYS ($xxxx + $0500).

    • Star Cruncher, a simple RLE-packer with fast load and save

      • NOTE: Sorry for the name "cruncher", but, when I wrote it, I didn't know the difference between packers and crunchers.

    • Level Loader, loads a file compressed with Star Cruncher

    • Cruel Cruncher, a slightly modified and more user-friendly version of 2.0++

      • WARNING: Compressed files have difficulties with running under Action Replay.

    • Beeftrucker, a simple RLE-packer

      • WARNING: The depacker routine messes up zero page and so locks up the BASIC interpreter.

    • Time Cruncher V5, very good for levelpacking

    • Digital Packer, a more intelligent packer

    • Mini Compactor, another packer

    • Disk-Demon, the best disk editor ever

    • Dir Master, an excellent directory editor

      • WARNING: The software does not allocate the new sectors used for storing data, when you save the changed directory, so a BAM copier will ignore those sectors.

    • Disk Wizard, good for editing the directory or unformatting a disk

    • Duplicator, an excellent file copier

      • USAGE: Load with "...,8,1" through the boot software and it will autostart.

    • File Copier, a very fast file copier

      • WARNING: The software does not check sectors while reading, so bad sectors will not be re-tried but copied without any warning.

    • File Master, another file copier

    • Copy 236, copies files between floppy disk and tape

      • WARNING: The software can load and save in turbo format only.

    • Copy 235, another disk-tape file copier

      • NOTE: The software can load and save in normal and turbo format, as well.

    • 1 Minute Copy, copies a whole disk in a bit more than one minute, in three passes

    • 60 Second Copy, copies a whole disk in one minute, in three passes and a little

    • Hack'em Copies, disk copiers and nibble copiers all in one

      • USAGE: Load with "...,8,1" and it will autostart.

    • Nybble Master, a nibble copier

    • Fast Validate, validates a disk fast

    • Unscratch, undeletes deleted files

    • Autoboot, prepends a 3 block autobooter to a program file

    • Gigaload, a very fast disk turbo, distributed with GEOS

      • WARNING: The software cannot load files longer than about 190 blocks.

    • Most Access 2, another very fast disk turbo with track cache

    • ABC Turbo, loads and saves files in turbo format from/to tape

    • Tape Master 2, another tape turbo

    • Head Tester, a simple graphical aid to adjust the head of the datasette

    • REL File Copy, copies a relative file from one disk to another

      • LANGUAGE: German.

    • DOS Wedge, adds some simple disk commands to BASIC

    • Star Verify, compares two binary files on disk

  • The STA disk, side 2 (1995-02-08) [stadisk2.zip, 138.104 bytes]

    • The Last Ninja 2 Musics, a music compilation of my all-times-favorite game

      • USAGE: Load with "...,8,1" and it will autostart.

      • WARNING: Do not rename the file, otherwise it won't be able to load the musics and background screens.

    • Help Plus, a very good multi-purpose aid for programmers

    • Graphics BASIC, a graphical BASIC extension

    • Simons's BASIC, a multi-purpose BASIC extension

    • Profi Assembler Plus, the full version

      • USAGE: Start with SYS 32768.

    • Profi Assembler, small version

      • NOTE: The assembler of Help Plus can be overlaid with this one.

    • Profi Mon, a simple monitor

    • Hash Mon, an excellent monitor

      • USAGE: Load with "...,8,1", it will autostart and relocate itself to the memory address entered.

    • Future Composer, a music editor for beginners

    • Wizzine Sinus, a good sinus editor

    • 1x1 Char Editor, 2x2 Char Editor and 3x3 Char Editor, character editors for different character sizes

    • Joystick Check, a simple software to check your joysticks in both ports
