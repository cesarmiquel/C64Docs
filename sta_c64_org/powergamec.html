Getting power from the game port

Construction

You need the following parts to build this cable:

  • One solderable 15-pin male plug (PC game plug)

  • A shielded 1-wire or unshielded 2-wire cable of about 20 centimeters

Solder one end of the cable onto the game plug. You should have the following connections:



     Cable        Game plug   


      GND          4, GND     


      +5V          1, +5V

Circuit diagram
