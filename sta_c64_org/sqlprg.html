Useful SQL software

last updated on 2010-08-15

Microsoft SQL

All software below are for MS-SQL 7.0 and 2000; they may or may not work on older and newer versions.

SQL scripts

  • FixUsers (2005-06-01), drops "phantom" users – those not connected to any login – from a database (© by Joe Forster/STA, 2002-2005) [fixusers.zip, 863 bytes]

  • ListObj 0.12 beta (2004-07-28), lists the definitions and contents of tables, views and stored procedures, including encrypted ones (© by Joe Forster/STA, 2004) [listobj.zip, 38.012 bytes]

    • WARNING: To use this from the command line, you will also need "sed" and "tr" from GnuWin32 or UnxUtils.

    • CREDIT: Parts of this software have been taken from sp_decrypt_7.

  • sp_decrypt_7 (2002-06-26), decrypts the definitions of encrypted views and stored procedures (© by Martin Rakhmanov aka jimmers, 2002) [sp_decrypt_7.zip, 3.022 bytes] (original download)

    • ALTERNATIVE: ListObj can do what this software can and more.

    • WARNING: The script must be run in MS-SQL 2000 as it uses the 64-bit "bigint" type during decryption. However, it can only decrypt the definition of objects that have been encrypted by MS-SQL 7.0.

External software

  • dSQLSRVD 1.1 (2001-06-27), decrypts the definitions of encrypted views and stored procedures (© by dOMNAR, 2001) [dsqlsrvd.zip, 1.045.555 bytes]

    • ALTERNATIVE: ListObj can do most of what this software can and more.

    • NOTE: This software connects to both MS-SQL 7.0 and 2000 and supports the decryption of object definitions encrypted by either MS-SQL version.
